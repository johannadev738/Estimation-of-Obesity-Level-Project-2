## 📊 Project Presentation

Course: CAP-4077 Knowlegde Disc. & Data Mining 
Team: Aaron Cole, Martin Patterson, Johanna Devilme

Description: 
Collaborative group project focused on buiilding and optmizing a machine learning model to predict obesity levels based onn eating habits and physical condition data.
By implementing a neural network using Scikiit-learn's MLPClassifier and conducted multiple experiments to evaluate model performance. We analyze the impact of network architechture, activation functions,
learning rates, and regularizatiion techniques to improve accuracy and generalization. The final model demonstrated strong predictive perfomeance and confirmed that obesity classification is a non-linear
problem requiring careful model tuning.
